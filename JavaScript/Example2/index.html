Coding Exercise

As part of building a form to be used in a time-reporting task your manager asks you to create a date-picker widget. Your date-picker widget should include the following features:

  • Allow the user to enter dates manually
  • Allow the user to pick from a set of dates
  • Confine date selection to within 1 year of today's date

Your code should incorporate these characteristics:

  • Design Elegance
  • Small code footprint
